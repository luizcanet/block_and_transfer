Block and Transfer
==================

Drupal 7 feature that adds the option to transfer content from a blocked account
to another.

Dependences
-----------

- VBO (https://www.drupal.org/project/views_bulk_operations)
- Rules (https://www.drupal.org/project/rules)
- Features (https://www.drupal.org/project/features)

Known Issues
------------

Don't work as expected with Overlay enabled.
